# ![Magic!](https://github.com/Red-Warlock/Presto-Change-O/blob/main/resources/images/magic-wand-64x64.png) Presto Change-O

 A GUI-based program for batch converting audio files from one format to another. Made primarily in Python. This program is a work in progress.
